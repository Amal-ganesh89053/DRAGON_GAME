# 🐉 Dragon Game

A browser-based endless runner game built using HTML, CSS, and JavaScript.

## Features
- Jump controls using arrow keys
- Obstacle avoidance
- Score tracking
- Collision detection
- Dynamic difficulty
- Background music

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How to Run
1. Clone the repository
2. Open index.html in a browser
